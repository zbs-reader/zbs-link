EPUB из Markdown

Локальный инструмент для сборки .epub из .md с обложкой, метаданными книги и общим блоком META_DATA для вашей читалки.

Как лучше

Если сайт с обложкой не даёт скачать картинку по ссылке, просто загрузите её файлом сюда. Для EPUB это самый надёжный вариант.

META_DATA блок

Блок будет вставлен в описание в общем формате метаданных: <!-- META_DATA_START -->, потом строки вида team: ... и boosty: ..., затем <!-- META_DATA_END -->.
